Introduction to PostGIS
=======================

Getting Started
---------------

* This workshop uses a `data bundle <http://s3.cleverelephant.ca/postgis-workshop-2018.zip>`_. Download it and extract to a convenient location.

* The **presentation deck** that goes with this workshop is available `on Google Slides <https://docs.google.com/presentation/d/1qYXdeCIymLl32uoAHvAPrp1r-hK-_4Z8InG7sHEo6vc/edit>`_.

* The **cheat sheet** of SQL commands that are run in this workshop is `available here <workshop-sql.txt>`_.


Data Bundle
-----------

Inside the data bundle, you will find:

**data/**
  a directory containing the data files we will be loading

**printing/**
  a directory containing the exercise pages and data model pages
